🎯 游戏目标

你扮演一名资本家，在三个历史纪元中做出关键决策。目标是积累财富，但要注意社会矛盾的激化——当矛盾达到100时，无产阶级革命爆发，游戏结束！

📊 核心指标

  • 财富：你的资本，初始100。破产（≤0）则游戏结束。
  • 社会矛盾：社会不满程度，初始30。达到100则革命爆发。
  • 技术力：科技水平，影响生产效率和某些事件的选项。

🌐 社会关系

  • 工人信任：过低（≤10）会触发「总罢工」紧急事件。
  • 政府支持：过高（≥90）会获得「政策红利」。
  • 媒体舆论：影响公众形象和股市情绪。
  • 竞争压力：来自竞争对手的压力。

🛤️ 路线倾向

  • 保守路线（⚔️）：追求短期利润，可能激化矛盾。
  • 技术路线（🤖）：通过技术进步解决问题。
  • 改良路线（🕊️）：调和劳资关系，缓和矛盾。

🎲 运气系统

每次选择后投掷D20骰子：出1-5点会有额外惩罚，16-20点会有额外奖励。这为游戏增加了Roguelike的随机性。

📈 股市系统

每回合可以在底部股市面板买卖股票。股价会根据当前事件和你的选择波动。低买高卖可以加速财富积累。

📚 认知碎片

达成特定条件可解锁认知碎片，多周目时可以携带碎片入场，获得额外加成。收集全部碎片可解锁成就「熟读《资本论》」。

🏆 结局

  • 真结局：技术极度发达 + 矛盾极低 → 自由人联合体
  • 普通结局：改良路线 + 矛盾低 + 财富高 → 福利国家的乌托邦
  • 坏结局：高矛盾 + 高财富 → 技术封建主义
  • 革命结局：矛盾达到100 → 无产阶级革命爆发

🎮 资本：轮回与破局

一款基于马克思主义理论的 Roguelike 教育游戏

📖 游戏背景

你扮演一名资本家，在资本主义发展的三个历史纪元中做出关键决策。你的目标是积累财富，但每一次选择都会改变社会矛盾、技术力，以及你与四大社会群体之间的关系。

🎯 核心玩法

  • 财富 / 社会矛盾 / 技术力：三大核心指标，决定你的命运
  • 社会关系：工人信任、政府支持、媒体舆论、竞争压力，会触发特殊紧急事件
  • 卡牌式选项：每个事件提供 3 种路线选择（保守 / 技术 / 改良），hover 可查看详细后果
  • 滑块决策：关键事件支持自由调节投资/替代/环保比例
  • D20 骰子：每次选择后掷骰，运气会修正最终结果
  • 股市小游戏：每回合可在常驻股市面板买卖股票，股价与事件联动
  • 认知碎片：达成特定条件解锁马克思主义理论碎片，多周目可携带入场

🏛️ 三大纪元

纪元一 原始积累 圈地运动、黑奴贸易、工厂法案
纪元二 工业垄断 经济危机、世界大战、反垄断法
纪元三 AI 数字 通用人工智能、气候危机、数字垄断
