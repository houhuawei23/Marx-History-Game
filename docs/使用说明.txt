《资本：轮回与破局》游戏使用说明
========================================

🎮 游戏简介
这是一款基于马克思主义理论的Roguelike教育游戏，通过模拟资本主义发展历程，
展示唯物史观、剩余价值理论等核心原理。

📁 文件说明
├── index.html          # 游戏主界面
├── style.css           # 游戏样式文件
├── game.js             # 游戏逻辑文件
├── 启动游戏.bat         # Windows批处理启动器（推荐）
├── 启动游戏.ps1         # PowerShell启动器
├── 打包游戏.py          # 打包工具（创建独立exe）
├── 创建桌面快捷方式.bat   # 创建桌面快捷方式
└── 使用说明.txt         # 本文件

🚀 一键启动方式（推荐）

方法1：双击启动（最简单）
  1. 双击运行 "启动游戏.bat"
  2. 游戏将自动在浏览器中打开
  3. 按Ctrl+C或关闭窗口停止游戏

方法2：创建桌面快捷方式
  1. 双击运行 "创建桌面快捷方式.bat"
  2. 桌面将出现游戏图标
  3. 双击桌面图标即可启动游戏

方法3：使用PowerShell
  1. 右键点击 "启动游戏.ps1"
  2. 选择"使用PowerShell运行"
  3. 游戏将自动启动

🎯 高级使用

打包为独立可执行文件：
  1. 确保已安装Python和PyInstaller
  2. 运行 "打包游戏.py"
  3. 选择打包方式
  4. 生成独立的exe文件，可在任何Windows电脑运行

手动启动（开发者）：
  1. 打开命令行
  2. 进入游戏目录
  3. 运行: python -m http.server 8000
  4. 浏览器访问: http://localhost:8000

🛠️ 系统要求
- Windows 7/8/10/11
- Python 3.6+（一键启动器需要）
- 现代浏览器（Chrome/Firefox/Edge等）

❓ 常见问题

Q: 启动时提示"未检测到Python"
A: 请从 https://www.python.org/downloads/ 下载安装Python

Q: 端口8000被占用
A: 启动器会自动尝试其他端口，或手动修改脚本中的端口号

Q: 游戏界面显示异常
A: 请使用现代浏览器，并确保网络连接正常

Q: 如何重新开始游戏
A: 游戏结束后点击"重新开始游戏"按钮

📞 技术支持
如有问题，请检查：
1. Python是否正确安装
2. 防火墙是否阻止本地服务器
3. 浏览器是否支持现代Web技术
